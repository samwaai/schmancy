Schmancy Details Component

Material Design 3 Expansion Panels with improved interactions

Default Variant

Material Design 3 introduces dynamic color, updated components, and new accessibility features. The design system emphasizes personal expression while maintaining usability.

Key features include:

  • Dynamic color theming
  • Updated motion and interaction patterns
  • Improved accessibility
  • Enhanced component flexibility

Ripple effects provide visual feedback at the point of interaction. They emanate from the touch or click point and expand outward, creating a natural and intuitive response.

Outlined Variant

Component Features:

  • Minimum touch target: 48px (mobile) / 56px (desktop)
  • Animation duration: 250ms with cubic-bezier easing
  • State layer opacity: 8% (hover), 12% (pressed)
  • Border radius: 12px (M3 medium component)

This component is built with accessibility in mind:

  • Full keyboard navigation support
  • ARIA-compliant markup
  • Visible focus indicators
  • Screen reader friendly

Filled Variant

The component uses Lit Element with Material Design 3 principles:

  • Shadow DOM for style encapsulation
  • Reactive properties for state management
  • Custom CSS properties for theming
  • Optimized render cycles

The component responds to various interaction states:

  • Hover: Subtle state layer appears
  • Focus: Primary color outline with offset
  • Pressed: Darker state layer and ripple effect
  • Keyboard: Enter/Space keys toggle expansion

Elevated Variant

Material Design 3 uses elevation to create visual hierarchy:

  • Level 0: No elevation (flat)
  • Level 1: Subtle shadow (collapsed state)
  • Level 2: Increased shadow (expanded state)
  • Smooth transition between elevation levels

The component is optimized for performance:

  • Hardware-accelerated animations
  • Efficient render updates
  • Minimal reflows and repaints
  • Lazy content rendering when expanded

This panel starts in the expanded state using the open attribute.

You can control the initial state programmatically or through the attribute.
