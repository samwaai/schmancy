Schmancy Boat - Draggable Demo

The boat component is now draggable! Try the following features:

  • Click and hold the header area to drag the boat around
  • The boat position is saved in localStorage and persists after page refresh
  • The cursor changes to a grab hand when hovering over the header
  • While dragging, the boat becomes slightly transparent with enhanced shadow
  • The boat automatically determines its anchor point (corner) based on position
  • Smooth animations during state transitions (minimized/expanded)
  • High z-index (9999) ensures it floats above other UI elements

How it works:

The dragging functionality is implemented using RxJS observables for smooth, reactive behavior. When you drag the boat to a new position, it automatically calculates which corner of the viewport it's closest to and anchors itself there. This ensures the boat stays in the expected position even when the window is resized.

The position is automatically saved to localStorage, so your preferred boat location persists across browser sessions.
